# BONC backends

## Prerequisites

- GNU/Linux;
- A C++26 ready compiler, e.g. GCC 15;
- CMake;
- 3rd party libraries:
  - Boost (all is preferred, required components: program_options, regex, and other header-onlys);
  - libGMP (required)
  - nlohmann_json (required)
  - libCryptoMiniSAT5 (maybe optional)
- Gurobi Solver for solving LP-format MILP model.
